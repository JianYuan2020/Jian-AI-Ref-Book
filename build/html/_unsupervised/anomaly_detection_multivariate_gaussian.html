Multivariate Gaussian Distribution¶

  • Parameters: \(x, \mu \in \mathbb {R^{n}}\), \(\Sigma \in \mathbb {R^{nxn}}\) (covariance matrix)

  • \(p(x; \mu, \Sigma) = \frac {1}{\sqrt {(2\pi)^{n} |\Sigma|}} \exp {(-\frac {1}{2} (x -\mu)^{T} \Sigma^{-1} (x -\mu))}\)

  • Here \(|\Sigma|\) is the determinant of \(\Sigma\).

Parameter fitting:¶

\(\mu = \frac {1}{m} \sum_{i=1}^{m} x^{(i)}\)

\(\Sigma = \frac {1}{m} \sum_{i=1}^{m} (x^{(i)} - \mu) (x^{(i)} - \mu)^{T}\)

Anomaly detection with the multivariate Gaussian

  1. Fit model \(p(x)\) by setting \(\mu, \Sigma\)

  2. Given a new example \(x\), compute

    \(p(x) = \frac {1}{\sqrt {(2\pi)^{n} |\Sigma|}} \exp {(-\frac {1}{2} (x -\mu)^{T} \Sigma^{-1} (x -\mu))}\)

Flag an anomaly if \(p(x) < \epsilon\)

Relationship to Original Model¶

  • Original model: \(p(x) = p(x_{1}; \mu_{1}, \sigma _{1}^{2})\) * … * \(p(x_{n}; \mu_{n}, \sigma _{n}^{2})\)

  • Corresponds to multivariate Gaussian

\(p(x; \mu, \Sigma) = \frac {1}{\sqrt {(2\pi)^{n} |\Sigma|}} \exp {(-\frac {1}{2} (x -\mu)^{T} \Sigma^{-1} (x -\mu))}\)

  • where all elements in \(\Sigma\) is zero except on the diagonal line
