Linear Regression with Multiple Variables/Features¶

Let’s define:

  • \(n\) = number of features.

  • \(m\) = number of training examples.

  • \(x^{(i)}\) = input (features) of \(i^{th}\) training example.

  • \(x^{(i)}_{j}\) = value of feature \(j\) in \(i^{th}\) training example.

  • \(x^{(i)} = [ x^{(i)}_{1}; x^{(i)}_{2}; ... x^{(i)}_{j}; ... x^{(i)}_{n} ]\) - \(n * 1\) column vector.

  • \(X = [ (x^{(1)})^{T}; (x^{(2)})^{T}; ... (x^{(i)})^{T}; ... (x^{(m)})^{T} ]\) - \(m * n\) matrix.

Hypothesis¶

\(h_\theta (x) = \theta_{0} + \theta_{1} * x_{1} + \theta_{2} * x_{2} + ... + \theta_{j} * x_{j} + ... \theta_{n} * x_{n}\).

Let: \(x_{0} = 1\) and \(x^{(i)}_{0} = 1\).

\(x = [ x_{0}; x_{1}; x_{2}; ... x_{j}; ... x_{n} ]\) - \((n + 1) * 1\) column vector.

Parameters¶

\(\Theta = [ \theta_{0}; \theta_{1}; \theta_{2}; ... \theta_{j}; ... \theta_{n} ]\) - \((n + 1) * 1\) column vector.

Therefore:

\(h_\theta (x) = \theta_{0} * x_{0} + \theta_{1} * x_{1} + \theta_{2} * x_{2} + ... + \theta_{j} * x_{j} + ... + \theta_{n} * x_{n}\).

\(h_\theta (x) = \Theta^{T} * x\).

Cost Function¶

\(J(\Theta) = \sum_{i=1}^{m} (h_\theta (x^{(i)}) - y^{(i)})^2 / (2*m)\).

Gradient Descent¶

\(\theta_{j} = \theta_{j} - \alpha * \sum_{i=1}^{m} (h_\theta (x^{(i)}) - y^{(i)}) * x^{(i)}_{j}\).

Note here, \(x^{(i)}_{0} = 1\); \(j = 0 ... n\); and \(\Theta\) is a \((n + 1) * 1\) column vector;
